# Hurricane Data Analysis (1851–2014)

## Overview
This project explores historical hurricane and typhoon data from 1851 to 2014 using Python.

The main goal is to clean the dataset, explore key atmospheric variables, and identify patterns related to storm behaviour, especially the relationship between wind speed and pressure.

## Objectives
- Clean and prepare historical cyclone data
- Perform exploratory data analysis (EDA)
- Investigate relationships between key meteorological variables
- Extract meaningful insights from extreme weather data

## Tools Used
- Python
- Pandas
- NumPy
- Matplotlib
- Jupyter Notebook

## Project Steps
1. Data loading
2. Data understanding
3. Data cleaning
4. Exploratory data analysis
5. Key insights and conclusion

## Project File
The full notebook is available in this repository: `MelissaLopes_HurricaneProject.ipynb`

## Key Insights
- Lower atmospheric pressure is generally associated with higher wind speeds
- The dataset shows a wide variation in storm intensity
- Extreme values highlight the presence of intense cyclone events

## Why this project matters
This project combines data analytics techniques with meteorology domain knowledge, showing how raw climate data can be transformed into meaningful insights.
